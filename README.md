# Contract Event Watcher
Watches ERC-1155 contract events using Infura and updates information to MongoDB.

## Installation
1. `npm install`
2. `node app`

## Mainnet ERC-1155 (Enjin)
`0x8562c38485B1E8cCd82E44F89823dA76C98eb0Ab`

## Watched Events:
1. `Create`
2. `Melt`
3. `Mint`
4. `setURI`
5. `Transfer`
6. `UpdateName`
